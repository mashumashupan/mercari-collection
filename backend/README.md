## Get started

1. Create a Virtual Environment:
```
$ python -m venv ./venv
```

2. Activate the Virtual Environment:
```
$ source ./venv/bin/activate
```

3. Install Dependencies:
```
$ pip install -r requirements.txt
```

4. Run Flask Application:
```
$ Flask run
```

5. Test with cURL:
```
$ curl -X GET "http://127.0.0.1:5000/"
Hello, World!
```
